# ERPHome Documentation

## Overview
ERPHome es un proyecto gestionado en Angular y Node.js. Este documento proporciona una visión general del proyecto y sus componentes principales.

## Archivos Principales
### app.ts
Contiene la clase principal de la aplicación, que se encarga de inicializar el estado global del componente raíz (`App`).

### mi-perfil/mi-perfil.ts
Componente Angular que gestiona la pantalla de perfil del usuario. Incluye lógica para cargar y actualizar los datos del usuario.

## Componentes Adicionales
- `moduloAutenticacion/servicios/usuario-service`: Servicio que se encarga de la autenticación del usuario.
